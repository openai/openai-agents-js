## 概述

[OpenAI Agents SDK for TypeScript](https://github.com/openai/openai-agents-js)
让你以轻量、易用、几乎无抽象的方式构建具备智能体能力的 AI 应用。它是我们此前
智能体实验项目 [Swarm](https://github.com/openai/swarm/tree/main) 的面向生产的升级版本，同时也[提供 Python 版本](https://github.com/openai/openai-agents-python)。Agents SDK 只有一小组基础组件：

- **智能体（Agents）**：配备 instructions 和 tools 的 LLM
- **交接（Handoffs）**：允许智能体将特定任务委托给其他智能体
- **护栏（Guardrails）**：用于对智能体的输入进行校验

配合 TypeScript，这些基础组件足以表达工具与智能体之间的复杂关系，让你在没有陡峭学习曲线的情况下构建真实世界应用。此外，SDK 内置**追踪（tracing）**，可视化并调试你的智能体流程，支持评估、以及为你的应用微调模型。

## 为什么使用 Agents SDK

SDK 的两个核心设计原则：

1. 功能足够丰富，值得使用；同时基础组件足够少，易于上手。
2. 开箱即用；同时允许你精确自定义实际行为。

主要特性：

- **智能体循环（Agent loop）**：内置循环，负责调用工具、将结果回传给 LLM，并循环直到 LLM 完成。
- **TypeScript 优先（TypeScript-first）**：用语言自带能力来编排与串联智能体，无需学习新的抽象。
- **交接（Handoffs）**：在多个智能体之间进行协调与委派的强大能力。
- **护栏（Guardrails）**：与智能体并行运行输入校验与检查，失败则尽早中断。
- **函数工具（Function tools）**：将任意 TypeScript 函数变为工具，自动生成模式（schema）并通过 Zod 进行校验。
- **追踪（Tracing）**：内置追踪，可视化、调试并监控工作流，并可使用 OpenAI 的评估、微调与蒸馏工具。
- **实时智能体（Realtime Agents）**：构建强大的语音智能体，包含自动打断检测、上下文管理、护栏等。

