## 概述

[OpenAI Agents SDK for TypeScript](https://github.com/openai/openai-agents-js)
让你以轻量、易用、抽象极少的方式构建智能体应用。它是我们此前
面向智能体的实验项目 [Swarm](https://github.com/openai/swarm/tree/main) 的生产级升级版，并且[也提供 Python 版本](https://github.com/openai/openai-agents-python)。Agents SDK 提供一小组基础组件：

- **Agents**：配备 instructions 和 tools 的 LLM
- **Handoffs**：允许智能体将特定任务委派给其他智能体
- **Guardrails**：使输入可以在到达智能体前被校验

结合 TypeScript，这些基础组件足以表达工具与智能体之间的复杂关系，让你无需陡峭学习曲线即可构建真实应用。除此之外，SDK 内置了**追踪**，便于你可视化与调试智能体流程，并对其进行评估，甚至为你的应用微调模型。

## 使用 Agents SDK 的原因

该 SDK 遵循两条核心设计原则：

1. 功能足够丰富，值得使用；同时保持足够少的基础组件，易于上手。
2. 开箱即用效果佳，同时允许你精确定制具体行为。

主要特性包括：

- **智能体循环**：内置的智能体循环负责调用工具、把结果发送给 LLM，并循环直至 LLM 完成。
- **TypeScript 优先**：使用语言内置能力来编排与串联智能体，而无需学习新的抽象。
- **交接**：在多个智能体之间进行协调与委派的强大功能。
- **护栏**：与智能体并行运行输入校验与检查，若检查失败可提前中断。
- **函数工具**：将任意 TypeScript 函数变为工具，自动生成模式并使用 Zod 驱动校验。
- **追踪**：内置追踪可用于可视化、调试与监控工作流，并可使用 OpenAI 的评估、微调与蒸馏工具。
- **实时智能体**：构建强大的语音智能体，包括自动打断检测、上下文管理、护栏等。
