## 概述

[适用于 TypeScript 的 OpenAI Agents SDK](https://github.com/openai/openai-agents-js)
让您以轻量、易用、抽象极少的方式构建基于智能体的 AI 应用。它是我们此前
针对智能体的实验项目 [Swarm](https://github.com/openai/swarm/tree/main) 的生产级升级版本，并且[也可用于 Python](https://github.com/openai/openai-agents-python)。Agents SDK 仅包含一小套基本组件：

- **Agents**：配备 instructions 和 tools 的 LLM
- **Handoffs**：允许智能体将特定任务委托给其他智能体
- **Guardrails**：对输入进行验证

结合 TypeScript，这些基本组件足以表达工具与智能体之间的复杂关系，让您在没有陡峭学习曲线的情况下构建真实世界的应用。此外，SDK 内置**追踪**，可视化并调试智能体流程，并支持评估，甚至为您的应用微调模型。

## 使用 Agents SDK 的原因

该 SDK 的两条核心设计原则：

1. 功能足够丰富以值得使用，但基本组件足够少以便于快速上手。
2. 开箱即用体验优秀，同时允许您精细定制行为。

SDK 的主要特性包括：

- **Agent loop**：内置智能体循环，负责调用工具、将结果发送给 LLM，并循环直至 LLM 完成。
- **TypeScript 优先**：借助语言内置特性编排与串联智能体，无需学习新的抽象。
- **Handoffs**：强大的多智能体协调与委派能力。
- **Guardrails**：与智能体并行执行输入验证与检查，检查失败可提前中止。
- **Function tools**：将任意 TypeScript 函数变为工具，自动生成模式并通过 Zod 驱动验证。
- **Tracing**：内置追踪，用于可视化、调试与监控工作流，并可使用 OpenAI 的评估、微调与蒸馏工具。
- **Realtime Agents**：构建强大的语音智能体，包含自动打断检测、上下文管理、护栏等。
