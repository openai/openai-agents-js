## 概述

[OpenAI Agents SDK for TypeScript](https://github.com/openai/openai-agents-js) 让你以轻量、易用、极少抽象的方式构建智能体应用。它是我们之前智能体实验项目 [Swarm](https://github.com/openai/swarm/tree/main) 的生产级升级版本，同时[也提供 Python 版本](https://github.com/openai/openai-agents-python)。Agents SDK 仅包含一小组基础组件：

- **智能体（Agents）**：配备 instructions 和 tools 的 LLM
- **智能体即工具 / 交接（Agents as tools / Handoffs）**：允许智能体将特定任务委托给其他智能体
- **护栏（Guardrails）**：对智能体的输入进行验证

结合 TypeScript，这些基础组件足以表达工具与智能体之间的复杂关系，让你无需陡峭学习曲线即可构建真实世界应用。此外，SDK 内置了**追踪（tracing）**，用于可视化和调试智能体流程，并支持评估，甚至为你的应用微调模型。

## 使用 Agents SDK 的理由

该 SDK 的两个核心设计原则：

1. 功能足够实用，但基础组件足够少，便于快速上手。
2. 开箱即用，同时可精细自定义执行过程。

主要特性包括：

- **智能体循环（agent loop）**：内置循环处理工具调用，将结果回传给 LLM，并持续迭代直到任务完成。
- **TypeScript 优先**：使用原生 TypeScript 语言特性编排与串联智能体，无需学习新抽象。
- **智能体即工具 / 交接（Agents as tools / Handoffs）**：在多智能体间协调与分工的强大机制。
- **护栏（Guardrails）**：与智能体执行并行进行输入校验与安全检查，未通过时快速失败。
- **函数工具（Function tools）**：将任意 TypeScript 函数变为工具，自动生成模式并通过 Zod 驱动的验证。
- **MCP 服务器工具调用**：内置 MCP 服务器工具集成，使用方式与函数工具一致。
- **Sessions**：在智能体循环内维护工作上下文的持久化记忆层。
- **人工干预（Human in the loop）**：内置在多次运行中引入人类参与的机制。
- **追踪（Tracing）**：内置工作流可视化、调试与监控，并支持 OpenAI 全套评估、微调与蒸馏工具。
- **实时智能体（Realtime Agents）**：构建强大的语音智能体，具备自动打断检测、上下文管理、护栏等能力。

