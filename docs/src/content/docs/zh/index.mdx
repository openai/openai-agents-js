## 概述

[OpenAI Agents SDK for TypeScript](https://github.com/openai/openai-agents-js)
让您以轻量、易用、抽象极少的方式构建具备智能体能力的 AI 应用。它是我们此前智能体实验
[Swarm](https://github.com/openai/swarm/tree/main) 的可用于生产环境的升级版本，并且也[提供 Python 版本](https://github.com/openai/openai-agents-python)。Agents SDK 只包含一小组基本组件：

- **智能体（Agents）**：配备了 instructions 和 tools 的 LLM
- **交接（Handoffs）**：让智能体将特定任务委托给其他智能体
- **护栏（Guardrails）**：对传入智能体的输入进行校验

结合 TypeScript，这些基本组件足以表达工具与智能体之间的复杂关系，让您无需陡峭的学习曲线即可构建真实世界的应用。此外，SDK 内置了**追踪（tracing）**，便于可视化与调试智能体流程，并支持评估，甚至为您的应用微调模型。

## 为什么使用 Agents SDK

该 SDK 的两大设计原则：

1. 功能足够有用，但原语足够少，便于快速上手。
2. 开箱即用效果优秀，同时允许您精细定制具体行为。

主要特性包括：

- **智能体循环**：内置循环负责调用工具、将结果返回给 LLM，并循环直至 LLM 完成。
- **TypeScript 优先**：使用语言内建特性编排与串联智能体，无需学习全新抽象。
- **交接**：在多个智能体之间进行协调与委派的强大能力。
- **护栏**：与智能体并行执行输入校验与检查，若检查失败可提前中断。
- **函数工具**：将任意 TypeScript 函数变为 tool，自动生成 schema，并提供基于 Zod 的验证。
- **追踪**：内置追踪用于可视化、调试与监控工作流，并可使用 OpenAI 的评估、微调与蒸馏工具。
- **实时智能体（Realtime Agents）**：构建强大的语音智能体，支持自动打断检测、上下文管理、护栏等。
