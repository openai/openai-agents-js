## 概述

[OpenAI Agents SDK for TypeScript](https://github.com/openai/openai-agents-js)
让你以轻量、易用、极少抽象的方式构建智能体应用。这是我们此前智能体实验
[Swarm](https://github.com/openai/swarm/tree/main) 的面向生产的升级版，并[也提供 Python 版本](https://github.com/openai/openai-agents-python)。Agents SDK 只有一小组基础组件：

- **Agents**：配备 instructions 和 tools 的 LLM
- **Handoffs**：允许智能体将特定任务委托给其他智能体
- **Guardrails**：用于验证传入智能体的输入

结合 TypeScript，这些基础组件足以表达工具与智能体之间的复杂关系，让你无需陡峭学习曲线即可构建真实应用。此外，SDK 内置**追踪**，可视化与调试智能体流程，并用于评估、甚至为你的应用微调模型。

## 为什么使用 Agents SDK

SDK 遵循两条核心设计原则：

1. 功能足够丰富值得使用，但基础组件足够少以便快速上手。
2. 开箱即用效果好，同时允许你精准自定义实际行为。

主要特性包括：

- **Agent loop**：内置智能体循环，负责调用工具、将结果回传给 LLM，并循环直至 LLM 完成。
- **TypeScript 优先**：使用语言内建特性来编排与串联智能体，无需学习新的抽象。
- **Handoffs**：强大的多智能体协调与委派能力。
- **Guardrails**：与智能体并行执行输入校验，校验失败将提前中止。
- **函数工具**：将任意 TypeScript 函数转换为工具，自动生成 schema，并由 Zod 驱动验证。
- **追踪**：内置追踪用于可视化、调试与监控工作流，并可结合 OpenAI 的评估、微调与蒸馏工具。
- **实时智能体**：构建强大的语音智能体，包含自动打断检测、上下文管理、护栏等。
