## 概述

[OpenAI Agents SDK for TypeScript](https://github.com/openai/openai-agents-js)
让你以轻量、易用且抽象很少的方式构建智能体 AI 应用。它是我们此前
针对智能体的实验项目 [Swarm](https://github.com/openai/swarm/tree/main) 的生产级升级版本，同时也[提供 Python 版本](https://github.com/openai/openai-agents-python)。Agents SDK 仅包含一小组基础组件：

- **Agents**：配备 instructions 和 tools 的 LLM
- **Handoffs**：允许智能体将特定任务委派给其他智能体
- **Guardrails**：用于对智能体输入进行校验

配合 TypeScript，这些基础组件足以表达工具与智能体之间的复杂关系，让你无需陡峭学习曲线即可构建真实世界应用。此外，SDK 内置**追踪**，可视化并调试你的智能体流程，支持评估，甚至为你的应用微调模型。

## 为什么使用 Agents SDK

该 SDK 遵循两条核心设计原则：

1. 功能足够有用，但基础组件足够少，便于快速上手。
2. 开箱即用体验优秀，同时可精细自定义执行细节。

主要特性包括：

- **智能体循环（Agent loop）**：内置循环，负责调用工具、将结果返回给 LLM，并循环直至 LLM 完成。
- **TypeScript 优先**：利用语言内置能力编排与串联智能体，无需学习新的抽象。
- **交接（Handoffs）**：在多个智能体之间进行协调与委派的强大能力。
- **护栏（Guardrails）**：并行执行输入校验与检查，失败时可提前中止。
- **函数工具（Function tools）**：将任意 TypeScript 函数变为工具，自动生成模式，并由 Zod 驱动的校验。
- **追踪（Tracing）**：内置追踪，便于可视化、调试与监控工作流，并可使用 OpenAI 的评估、微调与蒸馏工具。
- **实时智能体（Realtime Agents）**：构建强大的语音智能体，包含自动打断检测、上下文管理、护栏等。

