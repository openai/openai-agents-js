## 项目设置与凭据

<Steps>

1. **创建项目**

   在本快速开始中，我们将创建一个可在浏览器中使用的语音智能体。如果你想体验一个新项目，可以试试 [`Next.js`](https://nextjs.org/docs/getting-started/installation) 或 [`Vite`](https://vite.dev/guide/installation.html)。

   ```bash
   npm create vite@latest my-project -- --template vanilla-ts
   ```

2. **安装推荐包**（需要 Zod v4）

   ```bash
   npm install @openai/agents zod
   ```

3. **生成客户端临时令牌**

   由于该应用将在用户浏览器中运行，我们需要一种安全方式通过 Realtime API 连接到模型。为此，可以使用应在后端服务器生成的[临时客户端密钥](https://platform.openai.com/docs/guides/realtime#creating-an-ephemeral-token)。出于测试目的，你也可以使用 `curl` 和常规 OpenAI API 密钥生成一个密钥。

   ```bash
   export OPENAI_API_KEY="sk-proj-...(your own key here)"
   curl -X POST https://api.openai.com/v1/realtime/client_secrets \
      -H "Authorization: Bearer $OPENAI_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "session": {
          "type": "realtime",
          "model": "gpt-realtime"
        }
      }'
   ```

   响应包含一个顶层 `value` 字段（以 `ek_` 前缀开头），以及生效的
   `session` 对象。建立 WebRTC
   连接时，请将 `value` 用作客户端密钥。该令牌有效期较短，因此后端应在需要时签发新的令牌。

</Steps>

## 语音智能体创建与连接

<Steps>

1. **创建你的第一个 Agent**

   创建新的 [`RealtimeAgent`](/openai-agents-js/openai/agents-realtime/classes/realtimeagent/) 与创建常规 [`Agent`](/openai-agents-js/zh/guides/agents) 非常相似。

   ```typescript
   import { RealtimeAgent } from '@openai/agents/realtime';

   const agent = new RealtimeAgent({
     name: 'Assistant',
     instructions: 'You are a helpful assistant.',
   });
   ```

2. **创建会话**

   与常规智能体不同，语音智能体会在 `RealtimeSession` 中持续运行并监听；该会话会持续处理对话以及与模型的连接。这个会话还会处理音频处理、中断，以及我们稍后会介绍的许多其他生命周期功能。

   ```typescript
   import { RealtimeSession } from '@openai/agents/realtime';

   const session = new RealtimeSession(agent, {
     model: 'gpt-realtime',
   });
   ```

   `RealtimeSession` 构造函数的第一个参数是 `agent`。这个智能体将是用户最先可以交互的智能体。

3. **连接到会话**

   要连接到会话，你需要传入之前生成的客户端临时令牌。

   ```typescript
   await session.connect({ apiKey: 'ek_...(put your own key here)' });
   ```

   这会在浏览器中通过 WebRTC 连接到 Realtime API，并自动配置麦克风和扬声器用于音频输入与输出。如果你是在后端服务器（如 Node.js）上运行 `RealtimeSession`，SDK 会自动使用 WebSocket 连接。你可以在[Realtime 传输层](/openai-agents-js/zh/guides/voice-agents/transport)指南中了解不同传输层的更多信息。

</Steps>

2. **启动并开始对话**
