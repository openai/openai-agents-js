## 支持的环境

OpenAI Agents SDK 在以下服务器环境中受支持：

- Node.js 22+
- Deno 2.35+
- Bun 1.2.5+

### 有限支持

- **Cloudflare Workers**：Agents SDK 可用于 Cloudflare Workers，但当前存在一些限制：
  - SDK 目前需要启用 `nodejs_compat`
  - 需要在请求结束时手动刷新追踪数据。[参见追踪](/openai-agents-js/zh/guides/tracing#export-loop-lifecycle)获取更多详情。
  - 由于 Cloudflare Workers 对 `AsyncLocalStorage` 的支持有限，某些追踪可能不准确
- **浏览器**：
  - 目前在浏览器中不支持追踪
- **v8 isolates**：
  - 如果使用带有正确浏览器 polyfill 的打包器，应该可以为 v8 isolates 打包 SDK，但追踪将无法工作
  - v8 isolates 尚未经过充分测试

## 调试日志

如果您在使用 SDK 时遇到问题，可以启用调试日志以获取更多运行信息。

将环境变量 `DEBUG` 设置为 `openai-agents:*` 以启用调试日志。

```bash
DEBUG=openai-agents:*
```

或者，您可以仅对 SDK 的特定部分启用调试：

- `openai-agents:core` — SDK 的主要执行逻辑
- `openai-agents:openai` — OpenAI API 调用
- `openai-agents:realtime` — Realtime Agents 组件
