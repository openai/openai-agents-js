## バージョニング

このプロジェクトでは、`0.Y.Z` 形式を使った、semantic versioning を少し調整したバージョニングを採用しています。先頭の `0` は、SDK がまだ急速に進化していることを示します。各コンポーネントは次のように増分されます。

## マイナーバージョン（`Y`）

beta としてマークされていない公開インターフェースに対する **破壊的変更** については、マイナーバージョン `Y` を増やします。たとえば、`0.0.x` から `0.1.x` への移行には、破壊的変更が含まれる場合があります。

破壊的変更を望まない場合は、プロジェクトで `0.0.x` バージョンに固定することを推奨します。

## パッチバージョン（`Z`）

破壊的ではない変更については、`Z` を増やします。

- バグ修正
- 新機能
- 非公開インターフェースの変更
- beta 機能の更新

## サブパッケージのバージョニング

メインの `@openai/agents` パッケージは、独立して使用できる複数のサブパッケージで構成されています。現時点では、各パッケージのバージョンは連動しており、1 つのパッケージでバージョンが上がると、ほかのパッケージも同様に上がります。`1.0.0` に向けて進む中で、この戦略を変更する可能性があります。

## 変更履歴

何が変更されたかを理解しやすくするために、各サブパッケージごとに変更履歴を生成しています。変更がサブパッケージ内で発生している場合があるため、変更の詳細については該当する変更履歴を確認する必要があるかもしれません。
