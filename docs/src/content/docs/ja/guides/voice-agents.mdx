音声エージェントは OpenAI の speech-to-speech モデルを使い、リアルタイムの音声チャットを提供します。これらのモデルはオーディオ、テキスト、ツール呼び出しのストリーミングに対応しており、音声・電話でのカスタマーサポート、モバイルアプリでの体験、音声チャットなどの用途に最適です。

Voice Agents SDK は [OpenAI Realtime API](https://platform.openai.com/docs/guides/realtime) 用の TypeScript クライアントを提供します。

### 主な特徴

- WebSocket または WebRTC で接続
- ブラウザでもバックエンド接続でも使用可能
- オーディオと割り込みの処理
- ハンドオフによるマルチエージェントのオーケストレーション
- ツールの定義と呼び出し
- モデル出力を監視するカスタムガードレール
- ストリーミングイベント向けのコールバック
- 同じコンポーネントをテキストエージェントと音声エージェントの両方で再利用

speech-to-speech モデルを使用することで、音声を文字起こししてモデルの動作後に再度音声へ変換し直す必要がなく、モデルのリアルタイム処理能力を活用できます。
