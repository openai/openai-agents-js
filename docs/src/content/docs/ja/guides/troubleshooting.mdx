## サポートされている環境

OpenAI Agents SDK は次のサーバー環境でサポートされています:

- Node.js 22+
- Deno 2.35+
- Bun 1.2.5+

### 限定サポート

- **Cloudflare Workers**: Agents SDK は Cloudflare Workers で使用できますが、現時点ではいくつかの制限があります:
  - 現在 SDK は `nodejs_compat` を有効化する必要があります
  - リクエストの最後にトレースを手動でフラッシュする必要があります。[トレーシング](/openai-agents-js/ja/guides/tracing#export-loop-lifecycle) を参照してください
  - Cloudflare Workers における `AsyncLocalStorage` のサポート制限により、一部のトレースが正確でない可能性があります
  - 外向きの WebSocket 接続は fetch ベースのアップグレードを使用する必要があります（グローバルの `WebSocket` コンストラクタは不可）。Realtime の場合は、`@openai/agents-extensions` の Cloudflare トランスポート（`CloudflareRealtimeTransportLayer`）を使用してください
- **ブラウザ**:
  - ブラウザでは現在トレーシングはサポートされていません
- **v8 アイソレート**:
  - 適切なブラウザポリフィルを備えたバンドラを使用すれば SDK を v8 アイソレート向けにバンドルできますが、トレーシングは機能しません
  - v8 アイソレートでの動作は広範にはテストされていません

## デバッグロギング

SDK で問題が発生している場合、デバッグロギングを有効にすると詳細を確認できます。

`DEBUG` 環境変数を `openai-agents:*` に設定してデバッグロギングを有効にします。

```bash
DEBUG=openai-agents:*
```

または、SDK の特定の部分にスコープを絞ってデバッグできます:

- `openai-agents:core` — SDK の主要な実行ロジック
- `openai-agents:openai` — OpenAI API 呼び出し
- `openai-agents:realtime` — Realtime Agents コンポーネント
