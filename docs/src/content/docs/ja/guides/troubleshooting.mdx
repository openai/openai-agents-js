## 対応環境

OpenAI Agents SDK は、以下のサーバー環境でサポートされます:

- Node.js 22 以降
- Deno 2.35 以降
- Bun 1.2.5 以降

### 制限付きサポート

- **Cloudflare Workers**: Agents SDK は Cloudflare Workers でも使用できますが、現時点では以下の制限があります
  - SDK には `nodejs_compat` を有効にする必要があります
  - リクエスト終了時にトレースを手動でフラッシュする必要があります。詳細は [トレーシングガイドを参照](/openai-agents-js/ja/guides/tracing#export-loop-lifecycle)
  - Cloudflare Workers では `AsyncLocalStorage` のサポートが限定的なため、一部のトレースが正確でない場合があります
- **ブラウザ**:
  - ブラウザでは現在トレーシングはサポートされていません
- **v8 isolates**:
  - 適切なブラウザ向けポリフィルを備えたバンドラーを使用すれば SDK を v8 isolates 向けにバンドルできますが、トレーシングは機能しません
  - v8 isolates での動作は十分にテストされていません

## デバッグログ

SDK で問題が発生した場合、デバッグログを有効にすると状況を詳しく把握できます。

`DEBUG` 環境変数を `openai-agents:*` に設定してデバッグログを有効にします。

```bash
DEBUG=openai-agents:*
```

また、特定のコンポーネントに絞ってデバッグすることもできます:

- `openai-agents:core` — SDK のメイン実行ロジック
- `openai-agents:openai` — OpenAI API への呼び出し
- `openai-agents:realtime` — Realtime Agents コンポーネント
