## デバッグ ログ

SDK はデバッグ ログ出力に [`debug`](https://www.npmjs.com/package/debug) パッケージを使用します。詳細ログを表示するには、`DEBUG` 環境変数を `openai-agents*` に設定します。

```bash
export DEBUG=openai-agents*
```

`@openai/agents` の `getLogger(namespace)` を使って、自分のモジュール向けの名前空間付きロガーを取得できます。

<Code lang="typescript" code={getLoggerExample} title="ロガーを取得" />

### ログ内の機微データ

一部のログには ユーザー データが含まれる場合があります。次の環境変数を設定して無効化できます。

LLM の入力と出力のログ記録を無効化するには:

```bash
export OPENAI_AGENTS_DONT_LOG_MODEL_DATA=1
```

ツールの入力と出力のログ記録を無効化するには:

```bash
export OPENAI_AGENTS_DONT_LOG_TOOL_DATA=1
```
